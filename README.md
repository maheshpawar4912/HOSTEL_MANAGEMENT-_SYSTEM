# Hostel Management System

A simple hostel management system built with Node.js, Express, and SQLite.

## Features

- Manage hostels, rooms, students, and allocations
- Simple web dashboard for CRUD operations
- SQLite database storage for easy setup

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the application:

```bash
npm start
```

3. Open the dashboard:

```text
http://localhost:3000
```

## Project Structure

- `server.js` - Express server and API endpoints
- `db.js` - SQLite database initialization and connection
- `public/index.html` - Web interface for the system
- `public/app.js` - Browser UI logic and API integration
- `package.json` - Project dependencies and startup script
- `.gitignore` - Ignore rules for database and dependencies
